# DELTA — Tactical Geospatial Intelligence

DELTA uses Groq's Llama 3.3 70B model to turn natural language descriptions into geographic coordinates. It's a proof of concept for AI-assisted geolocation.(And it works)

## How it works

1. You enter a description(After running main.py and heading to the link)
2. DELTA sends it to Groq's LLM
3. The model returns coordinates + confidence + analysis
4. DELTA generates a PDF report in the Static folder.
